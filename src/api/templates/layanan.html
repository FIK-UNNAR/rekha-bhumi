Ketentuan Layanan API

Rekha Bhumi Nusantara

Selamat datang di layanan API Rekha Bhumi Nusantara. Dengan mengakses, mengintegrasikan, atau menggunakan API kami, Anda ("Pengguna" atau "Pengembang") dinyatakan telah membaca, memahami, dan menyetujui untuk terikat dengan seluruh Ketentuan Layanan ini. Jika Anda tidak menyetujui salah satu poin di bawah ini, Anda tidak diperkenankan menggunakan layanan kami.

1. Penyediaan Data Apa Adanya ("As-Is")

Seluruh data, informasi, kode wilayah, dan layanan penunjang yang disediakan melalui API Rekha Bhumi Nusantara diberikan kepada Pengguna atas dasar "As-Is" (apa adanya) dan "As-Available" (tersedia sebagaimana adanya). Kami tidak memberikan jaminan dalam bentuk apa pun, baik tersurat maupun tersirat, mengenai keakuratan mutlak, kelengkapan, keandalan, atau kesesuaian data untuk tujuan tertentu dari Pengguna.

2. Komitmen Perbaikan dan Pembatasan Tanggung Jawab

Kami berkomitmen penuh untuk menjaga kualitas data. Apabila ditemukan kesalahan teknis, kekeliruan data wilayah, atau gangguan sistem, pengelola Rekha Bhumi Nusantara akan berupaya melakukan perbaikan dengan segera setelah laporan diterima.

PENGECUALIAN TANGGUNG JAWAB HUKUM: Mengingat proses perbaikan memerlukan waktu dan penyesuaian teknis, pengelola Rekha Bhumi Nusantara secara hukum TIDAK DAPAT DIGUGAT ATAU BERTANGGUNG JAWAB atas segala bentuk kerugian, baik materiil maupun immateriil, kehilangan keuntungan, gangguan bisnis, atau dampak operasional yang timbul pada aplikasi/website Pengguna akibat adanya kesalahan data atau tidak tersedianya layanan API untuk sementara waktu.

3. Kebijakan Penggunaan yang Adil (Fair Use)

Pengguna wajib menjaga keamanan kredensial (jika menggunakan API Key) dan dilarang keras melakukan tindakan yang dapat membebani atau merusak infrastruktur server kami, termasuk namun tidak terbatas pada serangan DDoS, modifikasi ilegal, atau tindakan forceful scraping tanpa izin tertulis.

4. Perubahan Layanan dan Ketentuan

Kami berhak untuk mengubah, memperbarui, atau menghentikan sebagian fitur API atau Ketentuan Layanan ini sewaktu-waktu demi meningkatkan performa atau menyesuaikan dengan regulasi hukum yang berlaku di Indonesia.
