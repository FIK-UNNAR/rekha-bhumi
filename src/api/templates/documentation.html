Kode dan Data Wilayah Administrasi Pemerintahan ▾

Aplikasi Rekhabhumi menggunakan acuan resmi Kode dan Data Wilayah Administrasi Pemerintahan yang dikeluarkan oleh Kementerian Dalam Negeri Republik Indonesia (Kemendagri).

Format penulisan kode wilayah ini bersifat hierarkis dan dipisahkan oleh tanda titik (.) dengan struktur maksimal 4 blok angka -contoh formatnya: xx.xx.xx.xxxx. Blok angka ini merupakan kode administratif dari sebuah wilayah, dengan pembagiannya sebagai barikut:

1. Blok Pertama (kode Provinsi)

Blok pertama terdiri dari 2 digit angka yang menunjukkan identitas Provinsi.
Pada blok ini, awalan 1 dan 2 adalah provinsi yang ada di gugusan pulau Sumatra, awalan 3 untuk provinsi di gugusan pulau Jawa, awalan 5 untuk provinsi di gugusan pulau Bali dan Nusa Tenggara, awalan 6 untuk provinsi di gugusan pulau Kalimantan, awalan 7 untuk provinsi di gugusan pulau Sulawesi, awalan 8 untuk provinsi di gugusan pulau Maluku, dan awalan 9 untuk provinsi di gugusan pulau Papua.
Sebagai contoh: kode 11 adalah Provinsi Aceh, kode 21 adalah Provinsi Kepulauan Riau, kode 31 adalah Daerah Khusus Ibukota Jakarta, dan seterusnya.

2. Blok Kedua (kode Kota/Kabupaten)

Blok kedua terdiri dari 2 digit angka yang menunjukkan identitas Kota atau Kabupaten, yang berada pada provinsi yang ditentukan pada blok pertama. Pada blok ini, jika diawali dengan angka antara 0 sampai dengan 6 (01 sampai dengan 69) secara administratif wilayah itu adalah kabupaten. Sementara bila diawali 7 sampai dengan 9 (70 sampai dengan 99) secara administratif wilayah tersebut adalah kota.
Sebagai contoh: 11.01 adalah kode untuk kab. Aceh Selatan provinsi Aceh, 21.01 adalah kode untuk kab. Bintan provinsi Kepulauan Riau, 31.01 adalah kode untuk kab. Administrasi Kepulauan Seribu provinsi DKI Jakarta, 35.78 adalah kode untuk kota Surabaya provinsi Jawa Timur, dan seterusnya.

3. Blok Ketiga (kode Kecamatan)

Blok ketiga juga terdiri dari 2 digit angka yang menunjukkan identitas Kecamatan yang ada di Kota atau Kabupaten yang ditentukan blok kedua dan provinsi yang ditentukan pada blok pertama.
Sebagai contoh: 11.01.01 adalah kode untuk kecamatan Bakongan yang ada di kab. Aceh Selatan provinsi Aceh, 35.78.22 adalah kode untuk kecamatan Gayungan yang ada di kota Surabaya provinsi Jawa Timur, dan seterusnya.

4. Blok Keempat (kode Kelurahan/Desa)

Blok keempat terdiri dari 4 digit angka yang menunjukkan identitas Kelurahan atau Desa, yang berada pada kecamatan -yang ditentukan oleh blok ketiga, pada kota atau kabupaten -yang ditentukan oleh blok kedua, dan pada aprovinsi -yang ditentukan pada blok pertama. Digit pertama dari kode ini menandakan status apakah wilayah tersebut secara administrasi adalah desa atau kelurahan. Jika berawalan 1, maka wilayah tersebut secara administratif adalah kelurahan dan jika berawalan 2, maka wilayah tersebut secara administratif adalah desa.
Sebagai contoh: 11.01.01.2001 adalah kode untuk desa Keude Bakongan kecamatan Bakongan kab. Aceh Selatan provinsi Aceh, 35.78.22.1004 adalah kelurahan Ketintang kecamatan Gayungan kota Surabaya provinsi Jawa Timur dan seterusnya.

Namun direncanakan untuk kedepannya, kami akan meminta setiap pengakses untuk mendaftarkan origin aplikasinya dan kami akan memberikan API-Key untuk dipasang pada setiap kali request yang dilakukan.

Kami akan memberitahukan di situs ini pada saat proses otentikasi ini akan dilakukan.

End-Point API ▾

Terdapat tiga buah end-point pada API versi {{version}} ini, dan semuanya menggunakan method GET. Based URL API ini berada di {{url_aplikasi}}. Semua reply yang dikembalikan oleh server berbentuk json.

HTTP code reply setiap akses end-point sesuai dengan HTTP code yang digunakan dalam standar RFC7231. Pada versi {{version}} ini digunakan:

  • 200 OK - untuk akses ke end-point yang sukses dan terdapat data yang di-query.
  • 400 BAD REQUEST - menandakan bahwa akses ke end-point tidak mengikuti format yang telah ditentukan. Silakan request disesuaikan dengan format yang ada.
  • 404 NOT FOUND - request yang dilakukan berhasil dilaksanakan, namun tidak teradapat data yang diinginkan.
  • 500 Internal Server Error - jika terdapat kesalahan dalam aplikasi dan/atau server. Kontak kami jika hal ini terjadi.

Berikut penjelasan ketiga end-point API:

GET {{url_aplikasi}}/ver

Mengambil status versi aplikasi dan versi data yang digunakan dalam API Rekha Bhumi Nusantara. Jika diketahui ada data yang lebih baru, mohon untuk memberitahu ke administrator untuk dilakukan update data.

GET {{url_aplikasi}}/ver

Response (200 OK)

[{"code":"200","message":{"aplikasi":"1.0-00","database":"Kepmendagri No 300.2.2-2138/2025 data rev-2. Thanks to Cahya DSN (https://github.com/cahyadsn)"},"request":"/ver","status":"SUCCESS","total":2}]
GET {{url_aplikasi}}/list/[kode_wilayah]

Mengambil seluruh wilayah yang berada di bawah kode wilayah yang disebutkan. Jika kode_wilayah tidak dicantumkan, maka akan dikembalikan kode dan nama wilayah provinsi se-Indonesia.
Data kode_wilayah yang dapat diisikan hanya sampai level Kecamatan (blok ketiga), karena di bawah kelurahan/desa sudah tidak terdapat lagi data resmi dari Kemendagri.

GET {{url_aplikasi}}/list/35.78.22

Response (200 OK)

[{"code":"200","message":[{"kode":"35.78.22.1001","nama":"Gayungan"},{"kode":"35.78.22.1002","nama":"Menanggal"},{"kode":"35.78.22.1003","nama":"Dukuh Menanggal"},{"kode":"35.78.22.1004","nama":"Ketintang"}],"request":"/list/35.78.22","status":"SUCCESS","total":4}]
GET {{url_aplikasi}}/wilayah/<kode_wilayah>

Mengambil data nama dari kode_wilayah yang diisikan. Reply dari API akan mengembalikan semua nama wilayah dari kode_wilayah yang diisikan sampai ke atas (provinsi).

GET {{url_aplikasi}}/wilayah/35.78.22.1004

Response (200 OK)

[{"code":"200","message":{"Kabupaten_Kota":"Kota Surabaya","Kecamatan":"Gayungan","Kelurahan_Desa":"Ketintang","Provinsi":"Jawa Timur"},"request":"/wilayah/35.78.22.1004","status":"SUCCESS","total":4}]
