{{nama_aplikasi}}

adalah Application Programming Interface (API) yang akan memberikan kode wilayah resmi Republik Indonesia berdasarkan Keputusan Menteri Dalam Negeri (KEPMENDAGRI) yang berlaku saat ini. Sistematika kode wilayah ini dibuat berdasarkan hirarki wilayah dari provinsi, kota/kabupaten, kecamatan, hingga desa/kelurahan, sehingga dapat diakses menggunakan javascript dan digunakan langsung dalam aplikasi web dengan pilihan berbentuk SELECT secara cascade.

Untuk penggunaan private atau on premises di internal institusi, aplikasi (beserta database-nya) bisa didapatkan secara bebas melalui https://github.com/FIK-UNNAR/Rekha-Bhumi
