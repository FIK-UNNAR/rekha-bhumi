Kebijakan Privasi Layanan API

Rekha Bhumi Nusantara

Nama Platform: API untuk Negeri
Nama Layanan: Rekha Bhumi Nusantara
Tanggal Berlaku: 5 Juni 2026
Status Dokumen: Resmi / Publik

Selamat datang di Rekha Bhumi Nusantara (bagian dari "API untuk Negeri"), sebuah platform penyedia layanan Application Programming Interface (API) yang dikembangkan dan dikelola oleh civitas academy Fakultas Ilmu Komputer - Universitas Narotama. Kami sangat menghargai privasi para pengembang, instansi, maupun individu yang menggunakan layanan kami. Kebijakan Privasi ini dirancang untuk membantu Anda memahami bagaimana kami mengumpulkan, menggunakan, memproses, dan melindungi data informasi yang Anda berikan selama proses pendaftaran dan penggunaan API Key.

1. Informasi yang Kami Kumpulkan

Untuk memastikan keamanan jaringan, validitas integrasi, dan pembatasan akses penyalahgunaan layanan, kami mengumpulkan data spesifik berikut dari Anda saat mengajukan permohonan akses API:

  • Alamat Email: Kami meminta alamat email aktif pengguna yang berfungsi sebagai identitas akun pengembang, sarana komunikasi teknis, pembaruan layanan, serta pengiriman notifikasi status verifikasi.
  • Domain Origin (URL Website): Kami mewajibkan Anda untuk mendaftarkan alamat domain asal (origin domain) yang akan melakukan pemanggilan (request) ke server API kami.
  • Metadata Koneksi & Log Server: Server kami secara otomatis merekam data teknis standar seperti alamat IP pengirim request, timestamp, tipe browser/user-agent, dan HTTP method demi kebutuhan audit keamanan sistem.

Semua data yang dibutuhkan untuk proses verifikasi dan penerbitan API Key ini dikumpulkan secara sukarela oleh pengguna saat pendaftaran dan akan kami verifikasi dengan proses yang sesuai. Kami tidak mengumpulkan data pribadi lain di luar yang disebutkan di atas, kecuali jika diperlukan untuk kepatuhan hukum atau keamanan sistem.

2. Mekanisme Verifikasi dan Penerbitan API Key

Sesuai dengan komitmen kami dalam menyediakan layanan yang aman dan bertanggung jawab untuk negeri, setiap permohonan akses tunduk pada prosedur verifikasi berikut:

  1. Pendaftaran Akun: Pemohon mengisi alamat email valid yang akan digunakan sebagai identitas akun pengembang.
  2. Pengisian Data: Pemohon mengisi Domain Origin spesifik yang akan dipasangi API.
  3. Proses Verifikasi Akun dan Domain: Sistem akan melakukan pemeriksaan validitas, reputasi, kepemilikan, dan kesesuaian informasi yang didaftarkan.
  4. Penerbitan Kredensial: Jika domain dinilai aman dan memenuhi standar kepatuhan layanan, sistem kami akan menerbitkan token unik berupa API Key.
  5. Pembatasan CORS (Cross-Origin Resource Sharing): API Key yang diterbitkan dikunci secara ketat dan hanya dapat digunakan secara sah oleh website dengan Domain Origin yang telah terverifikasi tersebut.

3. Penggunaan Informasi dan Data Anda

Informasi yang kami kumpulkan dan verifikasi semata-mata digunakan untuk tujuan berikut:

  • Memvalidasi hak akses dan mencegah penyalahgunaan API (seperti serangan DDoS, scraping ilegal, atau spamming).
  • Mengirimkan informasi krusial terkait maintenance server, perubahan struktur endpoint, atau pembaruan regulasi teknis.
  • Menyusun statistik performa penggunaan API secara agregat dan anonim tanpa mengungkap aktivitas spesifik dari pengguna akhir (end-user) website Anda.

4. Keamanan dan Perlindungan Data

Kami menerapkan standar keamanan teknis dan organisasional yang ketat untuk melindungi email, domain, dan rahasia API Key Anda dari akses tidak sah, perubahan, atau pengungkapan yang melanggar hukum. Pemilik API Key juga berkewajiban secara mandiri untuk menjaga kerahasiaan token yang diberikan dan bertanggung jawab penuh atas segala aktivitas pemanggilan data yang berasal dari domain terdaftar mereka.

5. Berbagi Data dengan Pihak Ketiga

Platform api.unnar.id berkomitmen penuh untuk tidak menjual, menyewakan, menukarkan, atau memberikan alamat email dan informasi domain pengembang kepada pihak ketiga mana pun untuk tujuan komersial atau periklanan tanpa persetujuan tertulis dari Anda.

6. Hak Pengembang (Penghapusan Akun & Pencabutan API Key)

Anda berhak untuk memperbarui informasi domain atau meminta penghentian layanan kapan saja. Jika Anda memutuskan untuk menghapus data email dan domain Anda dari sistem kami, API Key Anda akan secara otomatis dicabut (revoked) dan website terdaftar tidak akan bisa lagi mengakses data dari server kami.

7. Perubahan Kebijakan Privasi

Kami berhak melakukan pembaruan pada Kebijakan Privasi ini sewaktu-waktu guna menyesuaikan dengan perkembangan teknologi dan regulasi hukum yang berlaku di Indonesia. Setiap perubahan signifikan akan kami informasikan melalui broadcast ke alamat email yang terdaftar di sistem kami atau melalui pengumuman resmi di situs api.unnar.id.

8. Kontak dan Layanan Bantuan

Apabila Anda memiliki pertanyaan, kendala teknis verifikasi, atau keluhan terkait penerapan Kebijakan Privasi ini, Anda dapat menghubungi tim pengelola sistem di Universitas Narotama melalui saluran resmi yang tersedia pada platform.
